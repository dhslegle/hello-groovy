| 字符串名称                | 字符串语法     | 内差值支持 | 多行支持 | 转义字符 |
|----------------------|-----------|-------|------|------|
| Single-quoted        | `'…'`     |       |      | `\`  |
| Triple-single-quoted | `'''…'''` |       | ✓    | `\`  |
| Double-quoted        | `"…"`     | ✓     |      | `\`  |
| Triple-double-quoted | `"""…"""` | ✓     | ✓    | `\`  |
| Slashy               | `/…/`     | ✓     | ✓    | `\`  |
| Dollar slashy        | `$/…/$`   | ✓     | ✓    | `$`  |
